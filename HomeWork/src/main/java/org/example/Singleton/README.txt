Тип шаблона: Создающий
Название шаблона: Singleton

Описание:
Шаблон Singleton используется для того, чтобы гарантировать, что у класса есть только один экземпляр, и предоставить глобальную точку доступа к этому экземпляру.

Почему использован:
Этот шаблон используется, когда необходимо ограничить создание объектов определенного класса до одного экземпляра. Это полезно, например, для управления доступом к общему ресурсу, такому как конфигурация приложения или логирование, где нужно быть уверенным, что существует только один активный объект.
