Тип шаблона: Структурный
Название шаблона: Adapter

Описание:
Шаблон Adapter позволяет объектам с несовместимыми интерфейсами работать вместе. Он действует как мост между двумя несовместимыми интерфейсами.

Почему использован:
Этот шаблон использован, чтобы обеспечить совместимость между классами `MediaPlayer` и `AdvancedMediaPlayer`. Это позволяет клиентам использовать новый интерфейс (например, воспроизведение разных форматов медиафайлов) без необходимости изменять существующие классы, что делает код более гибким и масштабируемым.
