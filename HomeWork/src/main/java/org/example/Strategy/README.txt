Тип шаблона: Поведенческий
Название шаблона: Strategy

Описание:
Шаблон Strategy позволяет выбирать алгоритм поведения на этапе выполнения программы, инкапсулируя его внутри объекта и предоставляя клиенту возможность выбора конкретной реализации.

Почему использован:
Этот шаблон использован для реализации различных стратегий оплаты (`CreditCardPayment`, `PayPalPayment`) в классе `ShoppingCart`. Он позволяет легко добавлять новые способы оплаты и изменять их без изменения существующего кода, что улучшает расширяемость и поддерживаемость системы.
